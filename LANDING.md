## Description

AI agents are clever and forgetful. They scaffold 200 lines before you've read the first ten. They forget last Tuesday's decision the moment the session ends. They tell you "looks good" against criteria nobody wrote down. For a weekend toy that's fine. For a real product — especially in healthcare, fintech, anywhere you can't afford to be wrong — it's a slow leak.

**bencium-harness** is a small Claude Code plugin (9 slash commands, 2 auto-skills, 1 hook) that gives your agent the rhythm a real engineer works in: a one-page plan you approve before any code is written, a checklist you wrote that gates the deploy, a memory that survives `/clear`, and a postmortem loop that fires when something fails. Run `/bencium-init`, answer five questions, and ~60 seconds later you have a PRD, an architecture, a `tasks.md`, an `ACCEPTANCE.md`, and a `.harness/` folder that holds the project's brain across every session.

If you're new to agentic engineering, this is the on-ramp. The loop itself tells you what comes next — plan, build, verify, reflect, deploy — so you don't have to invent the workflow while also learning the tools. If you're already shipping with AI, it's the structure that turns clever prompting into a practice you can run on something real.
